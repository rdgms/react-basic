# React + Mobx


## Setup

### Install node
    
  Using terminal:
  
  Install NVM: `curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.1/install.sh | bash` 
  
  Install node : `nvm install 6.9.1`
  

### Install yarn

 `npm install -g yarn`

### Install dependencies

 `yarn`

## Use

 `yarn start`
